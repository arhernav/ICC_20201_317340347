Para compilar el codigo existen dos opciones
     1.-Si se cuenta con gradle colocarse en el directorio Practica06, escribir en  la terminal el comando "gradle build" para compilar y en el mismo directorio usar el comando gradle build para correr
     2.-Colocarse en el directorio Practica06/src/main/java, compilar primero Matrix.java por medio de "javac Matrix.java", compilar MatrixTests por medio de "javac MatrixTests.java" correr por medio de "java MatrixTests"
